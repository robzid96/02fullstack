Pengenalan HTML

HTML adalah struktur dasar dalam pembuatan website atau kerangka utama dalam sebuah website

Sejarah HTML

HTML dimulai dari kebutuhan Tim Berners-Lee di CERN untuk membuat sistem hypertext guna berbagi informasi penelitian, dan ia mengusulkan ide ini pada tahun 1989. HTML pertama kali diperkenalkan pada tahun 1991 dan versi pertamanya dirilis pada tahun 1993. Pengembangan standar HTML dikelola oleh World Wide Web Consortium (W3C), yang kemudian melahirkan standar HTML 4.01 dan versi terbaru saat ini, HTML5, yang dirilis pada tahun 2014.

Perkembangan dan Versi Penting

  • Awal Mula (1989-1991) : Tim Berners-Lee mengajukan konsep hypertext untuk mengatasi masalah berbagi dokumen penelitian di CERN, dan kemudian mulai mengembangkan HTML untuk tujuan ini.
  • HTML 1.0 (1993) : Versi pertama HTML dirilis, memperkenalkan tag-tag dasar untuk menyusun dokumen digital di internet.
  • HTML 2.0 : Disetujui sebagai standar pada 22 September 1995, menambahkan fitur seperti unggahan berkas berbasis formulir dan tabel.
  • HTML 3.2 : Diterbitkan pada 14 Januari 1996, menjadi perkembangan penting dalam bahasa markup web.
  • HTML 4.01 : Menjadi standar resmi pada Desember 1999 dan populer di tahun 2000-an, menyediakan tiga variasi yaitu Strict, Transitional, dan Frameset.
  • HTML5 : Versi paling baru, dirilis pada tahun 2014, memperkenalkan fitur-fitur semantik seperti < article >, < footer >, dan < header > yang membantu pengembang memahami makna konten, serta mendukung pemutar video dan audio secara native.

Mengapa HTML itu Penting ?

  • Fondasi Pengembangan Website : HTML adalah bahasa markup standar yang digunakan untuk membuat dan menyusun semua halaman web. Tanpa HTML, tidak ada struktur pada sebuah website.
  • Mudah Dipelajari : HTML relatif mudah dipelajari, menjadikannya titik awal yang baik bagi pemula yang ingin masuk ke dunia pemrograman web.
  • Jalur Menuju Teknologi Web Lainnya: Memahami HTML adalah langkah awal yang krusial untuk mempelajari teknologi lain seperti CSS (untuk desain) dan JavaScript (untuk interaktivitas), serta untuk teknologi hybrid seperti React Native untuk aplikasi mobile.
  • Mewujudkan Ide Kreatif : Anda dapat membangun website pribadi, portofolio, atau bisnis online sendiri, serta mendesainnya sesuai selera dan kebutuhan tanpa bergantung pada template.
  • Peluang Karier : Keterampilan HTML membuka banyak peluang karier di bidang teknologi, termasuk pengembangan web front-end dan sebagai pengembang lepas.
  • Memahami Cara Kerja Website : Mempelajari HTML memberikan pemahaman yang komprehensif tentang cara kerja halaman web dan internet secara umum.
  • Meningkatkan Keterampilan : Proses belajar HTML dapat melatih logika berpikir, kemampuan memecahkan masalah, dan meningkatkan kemandirian.
